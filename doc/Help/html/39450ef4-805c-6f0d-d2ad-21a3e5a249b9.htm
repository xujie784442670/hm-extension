      KeyEventArgs 字段

      字段

      Code 虚拟按键代码。 代码必须是 1 到 254 范围内的值。
      Count 按键次数
      IsDown 是否按下
      IsExtend 是否是扩展键(如功能键,数字键盘上的键)
      IsUp 是否抬起
      ScanCode 按键的硬件扫描代码。
