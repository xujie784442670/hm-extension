      ObjectExtensionGetFieldValueT 方法

      获取对象的字段值

      示例

      C#
      class Student{
    public string Name ;
    public int Age ;
}

var stu = new Student{Name="张三",Age=20};
stu.GetFieldValue<string>("Name").Println(); // 控制台输出: 张三
stu.GetFieldValue<int>("Age").Println(); // 控制台输出: 20
