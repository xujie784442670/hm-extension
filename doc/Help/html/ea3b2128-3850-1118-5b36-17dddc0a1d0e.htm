      StringExtensionPrint(String, String, String, Object) 方法

      将字符串输出到控制台,支持格式化

      示例

      C#
      "Hello World".Print(); // 控制台输出: Hello World
"张三".Print("姓名: "); // 控制台输出: 姓名: 张三
"25".Print("年龄: ", "岁"); // 控制台输出: 年龄: 25岁
"{0}".Print("今天温度: ", "℃", 25); // 控制台输出: 今天温度: 25℃
