      StringExtensionToQRCode 方法

      将字符串转换为QRCode二维码

      示例

      C#
      // 将字符串转换为二维码
Bitmap bitmap = "http://www.baidu.com".ToQRCode();
// 生成一个绿色的二维码,背景色为白色
Bitmap bitmap = "http://www.baidu.com".ToQRCode(Color.Green, Color.White);
// 生成一个带有LOGO的二维码
Bitmap bitmap = "http://www.baidu.com".ToQRCode(icon: new Bitmap("logo.png"));

      Definition

      命名空间： HmExtension.Standard.Extensions
      程序集： HmExtension.Standard (在 HmExtension.Standard.dll 中) 版本：1.0.0.4+5d24d1b219f68227cde1bc968a2d9ca10604ff48
      C#
      public static Bitmap ToQRCode(
	this string value,
	Color? darkColor = null,
	Color? lightColor = null,
	int pixelsPerModule = 5,
	ECCLevel level = ECCLevel.Q,
	bool drawQuietZones = true,
	Bitmap icon = null,
	int iconSizePercent = 15,
	int iconBorderWidth = 0,
	Color? iconBackgroundColor = null
)

      参数

      value  String
      当前字符串
      darkColor  NullableColor  (Optional)
      暗色 二维码像素方块的颜色 一般设置为Color.Black 黑色
      lightColor  NullableColor  (Optional)
      亮色 二维码的背景色 一般设置为Color.White 白色
      pixelsPerModule  Int32  (Optional)
      生成二维码图片的像素大小,默认为5
      level  ECCLevel  (Optional)
      容错等级
      • ECCLevel.L: – 大约 7% 的错误更正能力
      • ECCLevel.M: – 大约 15% 的错误更正能力。
      • ECCLevel.Q: – 大约 25% 的错误更正能力。
      • ECCLevel.H: – 大约 30% 的错误更正能力。
      drawQuietZones  Boolean  (Optional)
      静止区，位于二维码某一边的空白边界,用来阻止读者获取与正在浏览的二维码无关的信息 即是否绘画二维码的空白边框区域 默认为true
      icon  Bitmap  (Optional)
      二维码水印图标 默认为NULL ，加上这个二维码中间会显示一个图标
      iconSizePercent  Int32  (Optional)
      水印图标的大小比例 ，可根据自己的喜好设置
      iconBorderWidth  Int32  (Optional)
      水印图标的边框
      iconBackgroundColor  NullableColor  (Optional)
      水印图标的背景色

      返回值

      Bitmap
      二维码图片

      备注

      在 Visual Basic 和 C# 中，这个方法可以当成为类型 String 的实例方法来调用。在采用实例方法语法调用这个方法时，请省略第一个参数。请参考 扩展方法 (Visual Basic) 或 扩展方法 (C# 编程指南) 获取更多信息。
