      MessageEventArgs 字段

      字段

      Hwnd 其窗口过程接收消息的窗口的句柄。 当消息是线程消息时，此成员为 NULL 。
      IsDelete 指定是否已从队列中删除消息。 此参数的取值可为下列值之一
      Message 消息的标识符。 应用程序只能使用低字;高字由系统保留。
      Point 发布消息时的光标位置（以屏幕坐标表示）。
      Time 消息的发布时间。
