      AddRangeT(ICollectionT, IEnumerableT) 将指定的集合添加到当前集合中

      示例

      C#
      List<int> list = new List<int> {1, 2, 3};
List<int> list2 = new List<int> {4, 5, 6};
list.AddRange(list2);
// list: {1, 2, 3, 4, 5, 6}
