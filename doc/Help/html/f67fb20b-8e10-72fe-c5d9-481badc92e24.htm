      ObjectExtensionGetPropertyOrFieldValueT 方法

      获取对象的属性或字段值

      示例

      C#
      class Student{
    public string Name {get;set;}
    public int Age ;
}

var stu = new Student{Name="张三",Age=20};
stu.GetPropertyOrFieldValue<string>("Name").Println(); // 控制台输出: 张三
stu.GetPropertyOrFieldValue<int>("Age").Println(); // 控制台输出: 20
