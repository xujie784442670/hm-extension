      StringExtensionReplaceRegex 方法

      使用正则表达式替换字符串

      示例

      C#
      "Hello World".ReplaceRegex(); // 输出: ""
"123abc456".ReplaceRegex(@"\d", "#"); // 输出: "###abc###"
