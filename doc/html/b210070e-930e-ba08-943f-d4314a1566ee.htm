      ByteArrayExtensionToHexString 方法

      将字节数组转换为16进制字符串

      示例

      C#
      byte[] arr = {0x01,0x02,0x3,0x4};
string hex = arr.ToHexString("-"); // 01-02-03-04
string hex2 = arr.ToHexString(); // 01020304

      Definition

      命名空间： HmExtension
      程序集： HmExtension (在 HmExtension.dll 中) 版本：1.0.0.2 (1.0.0.2)
      C#
      public static string ToHexString(
	this byte[] value,
	string separator = ""
)

      参数

      value  Byte
      当前字节数组
      separator  String  (Optional)
      分割符,默认为空字符串

      返回值

      String
      16进制字符串

      备注

      在 Visual Basic 和 C# 中，这个方法可以当成为类型 Byte 的实例方法来调用。在采用实例方法语法调用这个方法时，请省略第一个参数。请参考 扩展方法 (Visual Basic) 或 扩展方法 (C# 编程指南) 获取更多信息。
